# Road Accident Analysis in India 🚦

## 📊 Project Overview
This Excel dashboard analyzes road accidents in India using Power Query and visualization techniques.

## 🔧 Tools Used
- Excel
- Power Query
- Pivot Tables
- Charts & Dashboarding

## 📌 Key Insights
- Weekend vs Weekday accidents
- State-wise accident distribution
- Time-based accident trends
